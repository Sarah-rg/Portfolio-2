UX/UI


Wireframes

I learned the basics of wireframes and linking pages by creating a basic design using Figma. This enabled me to use buttons and see a prototype of what a website could do without needing to code it.



I practiced copying the Hostelworld website onto Figma without links to see the layout of a page, and all of the elements that would be needed to create a website.



Using the knowledge gained from practicing, I created a low-fidelity wireframe on paper of what I would like my portfolio website to look like.



I developed my low-fidelity wireframe into a high(er)-fidelity wireframe on Figma, including a menu page and links to show how I would like my pages to interact.



UX/UI At Work

In my freelance role I update and edit a website using Wordpress. I needed to create a new page on the website to present an upcoming fundraising campaign that the charity will be taking part in. To help organise the page, I implemented what I learned at Clickstart to sketch out a lo-fi wireframe, to help me plan how to layout the page.
